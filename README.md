# Academic CV LaTeX Template

Plantilla en LaTeX para la creación de un currículum vitae académico, orientada a investigadores y estudiantes de posgrado. El proyecto define un conjunto de macros reutilizables para estructurar educación, experiencia, habilidades, publicaciones y proyectos, con un diseño limpio y consistente.

---

## Características

- Diseño académico limpio y estructurado
- Uso de FontAwesome para iconos
- Barras de progreso con TikZ
- Macros personalizadas para:
  - Secciones
  - Eventos académicos
  - Habilidades
  - Iconos con texto y enlaces
  - Códigos QR
- Compilación con pdfLaTeX
- Alta capacidad de personalización
